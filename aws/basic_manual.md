# sshの接続方法
## 基本コマンドの例
```
ssh -i ~/.ssh/udemy_sample.pem ec2-user@<IPaddress>
```
pem ファイルは秘密鍵ファイル
IPアドレスには、インスタンスのパブリックIPアドレスを指定
